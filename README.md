# タスク管理アプリ

PHP + MySQL + MVC構成で作成した、タスク管理・進捗管理アプリです。  
タスクの登録、一覧表示、工程メモ管理、工程日時編集、進捗更新、カレンダー表示に対応しています。

---

## アプリ概要

このアプリは、1つのタスクに対して複数の工程を設定し、  
各工程ごとにメモや予定日時を登録できるタスク管理アプリです。

主な用途は以下のようなケースを想定しています。

- 個人タスクの進捗管理
- 作業工程ごとの予定管理
- 締切付きタスクの可視化
- カレンダー形式での予定確認

---

## 公開設定
Web公開ディレクトリは `public` を想定しています。  
ブラウザからの入口ファイルは `public/index.php` です。

---

## 主な機能

- タスク追加
- タスク一覧表示
- タスクの詳細確認
- 工程数の設定
- 工程メモの登録
- 工程日時の編集
- 進捗の増減・完了・リセット
- タスク削除
- ホーム画面でのカレンダー表示
- 日付ごとのタスク確認

---

## 使用技術
